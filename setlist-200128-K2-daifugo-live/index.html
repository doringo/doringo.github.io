感想和吐槽

这次中到了E block。本来会担心看不到，但是好心人告诉我Eblock有个台阶（目测有三十多公分），其实要比前面的Cblock视线好很多，而且我大概在1.5排，基本都看得很清楚。

影アナ是大场。

M01 誰かの耳 / ☆江籠
M02 オネストマン / ☆高柳
M03 回遊魚のキャパシティ

穿着公演后半的衣服，从天使C是谁耳开始。谁耳好像没有怎么放在开头过，这算是一种有趣的开端。因为这首歌call有限，但是又能让人看着热血沸腾，是很有跌宕起伏的一首歌。算是最终铃声时期的K2的代表作。考虑到后面的大容量setlist，算是节省了一点体力吧（对于饭来说）。天使C真的开着很带劲。

然后是来自剧场女神公演的诚实人。这首歌总能让人想到储金。真的是首好歌。再加上鸟的煽り，感觉像把六年前的Team C和现在的K2跨越时空相连。尤其是日高当时没有上过剧场女神，现在却是K2的主力干将。

紧接着，一看到鸟在中间的三角阵型就知道是洄游鱼了。capacapacapacapacapacity。

第一段MC吐槽了日高双冠王和逆天的narup和苏打。日高真的是蠢死了噢。这样子算下来日高就只赢了一回噢。怎么能这么蠢啊啊。

前半8人的环节。

M04 君のc/w / ☆水野(桃)、大場(紫)、惣田(青)
M05 初恋よ、こんにちは / ☆片岡、青木、竹内
M06 Must be now / ☆北野、江籠、荒井、大場、日高
M07 誰にも言わないで / 白井
M08 僕たちは戦わない / ☆大場、北野、古畑、荒井、高柳、竹内、日高、江籠
M09 永遠プレッシャー / ☆大芝、ALL?
M10 風は吹いている / ☆古畑、ALL?

爱理这个cw我觉得听适合她的声线的，只不过两边配的都是年龄大十岁的大姐大们，估计爱理也会有压力。

narup的初恋你好，这个人选很好，不过我个人的第一反应还是回想了一下p和阿弥有没有什么瓜葛。（好像没有）

瑠华C的这首难波的单曲，个人私心舞蹈比后面的我们不战斗要帅多了（不过我们不战斗的歌比较好听）。虽然旁边几个人一直都在hea来hea去，但是还是有明显听到天使的声线，真的让人把持不住啊。

扣酱的solo，有一段时间各种经常听到，的确是最近都没有出现了，因为在RH不是投票范围内，所以也很难得有机会唱。算是给后面的成员准备一点休息的时间吧。

大场C的我们不战斗，歌还是好听的，不过这首歌MV的印象比较多，翻道对这个号称最难舞蹈的舞蹈就没有什么印象了。尤其是中间趴在地上跳的那一段，真的看得我一脸懵逼（什么都看不到）。

永远的压力，没想到rinka会选这首歌，但是从最开始的solo开始就觉得这首歌无论是声线、歌词等等各方面都适合极了她。

本来以为狐狸会来啥刺激的，来个大风吹，感觉虽然没有体现出她个人的特点，但是……大风吹也是好歌。

天使C的无意识，其实大家也都觉得无意识不是唱烂了嘛，天使是说想要选SKE的歌，毕竟入团八九年了，自家的意识还是很高，但是听了前两天天使和日高的观览车的广播，大家都很感谢天使选了无意识——因为不怎么需要练习，大家都可以把时间放在别的歌上面。

然后的革命部分真的很好玩，我真的笑cry。

中野大可爱和大芝骂人真的好解气啊。尤其是骂狐狸天使的时候，大可爱的语气真的是太好玩了。

后半扣酱爱理骂人也很有意思，后半的前辈都很狡猾会顶嘴，不过还是会被顶回去。

这段MC真的好好玩可以看10遍。

后半8首。

前半的must be now已经被日高帅得七荤八素了，没想到后半来的刺激更大。

彩夏的candy穿的是我的太阳的那套衣服，还是当年须田那套衣服可爱一点。

然后就到了我个人爆炸的时刻——中野日高荒井的zipper！

らぶりん在D3甄选的时候有在sr唱过zipper，穿着黑色水手服一边唱还一边脱外套，没想到我有一天能亲眼看到她唱！而且还是带着日高！我的妈呀！日高这种人八百年都不可能给她唱zipper好不好！

而且中野らぶりん真的好适合这首歌啊。无论是音色、舞蹈还是气质。可能因为看多了S的zipper，对兰兰那种zipper印象多了之后，看到这样纤长的肉体又感受到了别样的刺激。

在脱下的瞬间，她的身材真的让人血脉喷张。不愧是走过T台的人，我觉得日高已经很瘦了，但是和她比还是相形见绌。那腹部真的是让人把持不住。prpr千千万万遍。

而且就算我隔着几十米远，也能清晰地看到她腹部的完美线条，她真的没有让我事忘啊。

总之这个zipper是绝了，真的绝了。我愿意称中野爱理为最强。

我还没从zipper缓过来，狐狸鸟天使的太宰治又让我感受到新的刺激。

我磕这三个人也有一段时间了，她们能一起唱这个歌也算是对她们的关系的一种诠释。

鸟说其实她更喜欢第二段的歌词，但是因为编排的关系没有办法唱。我去查了一下：

  • 風はそれでも
    何も変わらず吹いていた
    スーパーの安売りのちらしが
    ひらひら　振り回されてた

  • 行けど夢など
    何(なん)にも見つからなかった
    夢なんか見てしまったら
    挫折が怖くなるだけ

  • なぜか涙も出やしないよ
    愛も枯れるもんなんだ
    今日を淡々と生きて
    考えないようにする

  • 太宰治を読んだか?と聞かれた
    君を友と呼ぼう
    残念なことに
    本に答えはなかった
    “目から鱗が落ちた”というような
    奇跡はないけど
    ただ　人生とは何か?
    語れる友ができた

又想起来她写的：【考え過ぎない、比べ過ぎない、無理はしない、君は君らしく】

这三个人之间是柔情也是亲情，是一种超越了很多表面关系的直达心灵的信赖。

这是一部关于K2的血泪史，是用这三个人的名字镌刻的，火红的历史。

欧西林的远距离海报。全程我都在看らぶりん。歌词挺有意思。

苏打第一次front的Vacancy，感觉这一两年披露的次数变多了，不过还是很罕见的一首歌。不过emmm这个歌词有点biao，就是O接着OBC接着O。

彩姬选了这首骂我吧达令让我喜出望外我对上一次这首歌披露的印象还是在五年前的rumi选拔，由芽C到一半突然发疯，还有就是第一次还是第二次午夜公演的时候真那带着由麻奈C这首歌，让我们汗颜又不知所措。这首歌是元K2拿到的原创歌，茉夏鸟的C，传承感满载。

然后是诚实人和candy、初恋你好之后的第四首剧场女神的歌——陨石。陨石这首歌最好玩的还是最后一段说悄悄话的部分，陨石这首歌的歌词也是特别好。

然后就到了全场的另外一个高峰——日高的蜂巢dance。这首歌我也是出来之后才这道这是8的歌，完全不知道名字，不知道怎么call，却能看得很high很开心。

日高又傻又可爱真的也是没药救了。她能选出一首这么适合K2的歌，也是她全身流淌的红色——正是K2的颜色啊。

本篇最后一首歌献给剧场女神narup。

安可首先是RESET，然后是16人蜡笔。分别是0start和两次波子汽水的缩影。最后以最浓厚K2色彩的DADA机关枪收尾。这不是K2最初的对歌，但是是K2的代表，是K2中的K2。

K2的历史，完整地写在大富豪地setlist里。每一个人都是K2历史。

live结束之后又聊了两句，拍大合照之后撤了。
