个人吐槽

平时这种小live我都不会特别写的。

不过这场我自己也在就写一写啦嘿嘿。

我站在比较靠后的地方，在风雨中举着绫巴的应援色。

在照片中隐隐约约看到了自己（嘘

17：10的时候SKE才开始，然而我已经站在雨中淋了两个小时了w

（早知道回酒店休息了，毕竟刚看完昼场青G）

其实我这次是冲着这个成员配置来的。

甚至是为了这个改了机票。

（其实早就想改了刚好给自己一个强有力的理由）

出演／北川綾、江籠、大場、小畑、惣田、高柳、古畑、熊崎、菅原、須田

这个配置不要更神一点好吗，基本上把当家花旦都搬出来了。想想这群人总选最差都是NG就很可怕啊！

M01 恋を語る詩人になれなくて / ☆北川

一开始的诗人就带动了情绪，仔细看一眼还是绫巴C更开心了。

（这个时候要把自己当作绫巴厨）

M02 チャイムはLOVE SONG / ☆熊崎、☆菅原

全手牵手最喜欢的歌。
居然mix还能喊错。
哎哎哎不管了总之现场听这首真的很棒！真的！
用一个词概括就是【跃动的青春】
无论是歌词还是旋律都炒鸡喜欢

M03 無意識の色 / ☆小畑

定番无意识小畑流w

M04 涙の湘南 / 北川綾、小畑、古畑、大場、熊崎

看到立麦我第一反应就是湘南。
果不其然就响起的海边的水声。
重要的是还有狐狸啊！狐狸的声音真的分辨率太高太有吸引力了！

（最近沉迷狐毒w）

M05 コスモスの記憶 / 高柳、惣田、須田、菅原、江籠

这首歌让我第一个想起的由芽。
联系上面的湘南，可以感受到这个歌单【秋】的主题，毕竟是秋祭り，我想还是迎合主题的

M06 狼とプライド / ALL（メ〜テレのキャラクター・ウルフィと一緒に）

这个狼真的是www
全员狼就很wwww
前半天使call后半绫巴call。
没想到前一天还在微博上发5期的同人文然后她们就在我面前跳狼w

M07 センチメンタルトレイン / ☆須田

要唱这个我都吓到了www
真的是莫名其妙进来的一首歌
不过至少还是秋天的氛围所以给个勉勉强强合格吧

M08 意外にマンゴー / ☆小畑

小畑流定番。
我最喜欢的是
「恋はいつでもきっとストイックがいい」
这一段。
唤起了我沉睡着的（或者说从来没有过的）少女心

M09 バズーカ砲発射! / ☆須田

又是一首充满情怀的歌！

M10 オキドキ / ☆古畑

虽然不感冒okdk
但是很high所以很ok

（居然是狐狸c我都没注意到）

M11 アイシテラブル! / ☆須田、☆小畑

毛衣开是yunana，嘛不是狐狸全部差评

終演

最后居然有出待ち。
虽然我是站在超级远的地方看的w
不过看到成员出来就挺感动的。
