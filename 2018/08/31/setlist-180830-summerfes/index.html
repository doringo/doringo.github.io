終演

好久没有写东西了，昨天这个setlist算是彻底把我炸出来了。

matome上超过200评论的讨论，褒贬不一的这个setlist，虽然有些细节还没有仔细看，但是总体来说对这个具有冲击力的setlist进行一个个人的点评吧。

整体来说，还是典型的【全体——unit——全体】格式，SKE以外的歌曲比例相当高。

要是评选top3看点的话

  • 宛如寸剧的进行形式

  • 歌唱力zeroposi成员的生唱

  • 村外曲的独特呈现

让我们从头看起。

M01 放課後レース / ☆須田、☆高柳

放课后赛跑是一首典型的S曲风的歌，而且还加入了S公演，基本上代表了早期S的风格，在之前rh100上排名也很高。一般来说很少当作全体曲。

但是在年初的士多胖公演中日高的放课后大受好评，也站稳了她的“最强同级生”的称号。再加上现在的S比起青春活力更加酷炫帅气，所以放课后扩展为全体也无可厚非。

不过，C换成场上的S成员或者年轻成员，比如椰岛、yunana和慈子感觉效果会好一点。放开头第一首歌也没能起到燃气氛的作用，所以这个安排个人感觉还是欠妥。

M02 ごめんね、SUMMER / ☆北川綾

对不起夏天经典老歌，玲奈位是狐狸，鸟原位，然后是崩卡，中规中矩。

M03 オキドキ

狐狸崩卡双C。不知道是我个人对okdk不感冒还是因为从UC公演到S公演听到吐，有鸟的煽り燃下气氛还可以。

M04 生意気リップス / 小畑、水野

当年博多双小的歌，给小野唱也挺不错。只不过整首歌挺无聊的，没什么特别的看点。

途中yunana的话筒有点小问题？所以说的话没怎么听清。

M05 炎上路線 / 白井、松村

BBQ当年是有唱过这首，但是把扣酱安排在这儿，我也和扣酱一样满脸黑人问号。

首先扣酱没有过特别炎上的发言，而且和BBQ是完全不一样的努力方式。这首歌说实话也不是很适合两个人出来唱，所以就有点莫名其妙的。

M06 ジッパー / 松本、北川綾→北川綾、佐藤、仲村

我觉得慈巴唱zipper完全贴合主题啊！完全ok啊！

甚至慈巴+佐藤我都觉得まあまあ还ok，但是中途把慈子换下场是什么操作。

个人觉得正确的玩法应该是这样：慈子唱完第一段副歌，佐藤插进来捣乱。这样的话慈子也不至于被晾在一边啊。

M07 TWO ROSES / 高柳、須田

w松井曲，这两位毫无爆点。如果可以的话我更愿意看狐狸×鸟或者真木×崩卡的版本。虽然说两个人一前一后进来，一直在选拔，也有team c的友谊，但是终究还是走着不一样的擦肩而过的路的，所以说感情上还是更接受上述两对。

「ちゅりクエストアワー セットリストベスト5 2018なちゅ」（生歌・生演奏）
M08 5位 プラネタリウム(大塚愛) / 杉山愛佳
M09 4位 カブトムシ(aiko) / 野島樺乃
M10 3位 打上花火(DAOKO × 米津玄師) / 古畑奈和

或多或少都有点小崩。这里表扬一下椰岛，发挥应该是最稳定的。狐狸自身技巧很好，稳定欠佳。

M11 2位 BANG!BANG!バカンス!(SMAP) / 大場、高柳、斉藤、須田、松村
M12 1位 睡蓮花(湘南乃風) / 内山、松本（ALL）

都在玩，不过像我第一次听这两首歌的有点小懵逼，不过挺high的就是了。

中间切换到桌子和店里的专场的主意很棒！

M13 夏よ、急げ！ / ☆古畑
M14 賛成カワイイ! / ☆野島、☆小畑

中规中矩的两首歌。

希望能够把椰岛放到更重要的位置。

M15 君の背中 / ☆大場

这首歌可有意思。

这是当时米娜伦在被谨慎的时候发售的akb的cd的歌。

歌的演唱人员是【除了被谨慎的大场美奈以外的AKB全员】。

当时谁都没想到米娜伦会在2014年移籍到SKE，在2018年的总选上一举斩下第八位扬眉吐气。

这是一个笑话，也是一段佳话。

补充：后来听说的：原来这首歌打算让大场center的，但是后来出事，所以变成了除了她以外的全体曲。
