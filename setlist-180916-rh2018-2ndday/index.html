#个人吐槽

49位 マツムラブ！ / 松村

昨晚宣布毕业，今早第一发就是松村爱，真实唏嘘不已啊。

48位 いきなりパンチライン / 選抜 ☆須田、アンダー荒井

胖鸡线这个排名算是很高了。
一定程度上也挺高的。

47位 未来とは? / ☆井上、上村、北川愛、坂本、杉山、仲村、野村、白井、水野、相川、浅井、井田、倉島、斉藤、野々垣

每次未来都是给年轻的孩子们唱的。
毕竟未来都是年轻人的。

46位 Parting shot / チームS ☆北川綾

这首队歌居然这么高!
可能有剧场的影响力吧。

45位 愛のために何を捨てる? / ☆古畑、荒井、江籠、北野、鎌田

这首歌是2专里狐狸c的歌。
能找到的剧场披露也只有去年的midnight。
2专里的这首还有zero pace都是比较电音风格的歌，虽然狐狸还是给人传统的jpop的感觉比较多，但是拥有一首这样充满个性的c我觉得对于狐狸来说也是很好的。
我个人炒鸡喜欢这首歌的歌词，之前看哪个介绍说是属于抖s的歌词。

44位 キスポジション / チームKⅡ ☆小畑

话题度颇高的一首歌

43位 ハート型ウイルス / ☆竹内、青木、山田

UC秋的unit。
今天重温UC公演的时候不由得再次感慨，sakipon真的是太可爱太可爱了。

42位 Escape / 北川綾、杉山、都築、野村、山内、内山、江籠、大場、北野、惣田、高柳、日高、古畑、井田、熊崎、斉藤、須田、キーボード：松本

慈子继承了东大人的键盘。
不知道是谁c，但是这个阵容我很喜欢，基本都是dance member。想看这些人上s公演。

41位 制服を脱ぎたくなって来た / ☆小畑、上村、野島、水野、末永、菅原

【制服又脱又穿】梗里面的脱！
7D2的宝贝们！

40位 Gonna Jump / ☆須田、北川綾、江籠、大場、太田、小畑、惣田、高柳、古畑、浅井、熊崎、白雪、菅原、高畑、谷、福士

乐乐不在，同队队长c。带了大朋友小朋友们一起跳起来。

无意识的cw，是17年总选80到100的人们。

36位 お手上げララバイ / 斉藤

在剧场看过的表示，真木男真的太帅了。
太帅了真的。

熊版也很酷。想看日高版。

35位 放課後レース / チームS ☆北川綾、アンダー大谷

我从这首歌的1分半钟广播音源就开始喜欢这首歌。
喜欢它的欢快、喜欢它的青春、喜欢它的mv。
作为现s的第一首队歌，当时就有经常在剧场披露，加入了公演安可的歌，却不曾听腻过。
唯一一点小小的不满就是，这首歌一定要穿制服才是最纯粹的味道，不过现在的S是缺乏了制服的元素了，有点可惜，明明大家都那么年轻。

34位 目が痛いくらい晴れた空 / 江籠、古畑

14年目痛3位，还有八九个人一起唱这首歌。
今年就只剩下两个人了。
每年美滨的5期吃冰比赛今年也没有进行。
当一个期的人少到一定程度的时候，更多地开始关心conbi内部或者是个人了呢。

五期是最后一期站上过旧剧场的成员，不由得感慨一下了。

33位 僕は知っている / ALL

珠理奈预测的第一。
ske有几首每次周年纪念放到最后来唱的歌，比如说8单的「今日までのこと、これからのこと」，还有13单的「ずっとずっと先の今日」，到17单的我知道，而且又是ske第一部纪录片的主题曲，包含了太多过去。
每次在公演上看到带头喊ske的人越来越年轻就不由得感慨了。

33位 兆し / チームKⅡ

波子汽水是多少人回忆里的青春。
去年unit con的安可，理沙子和全团人一起唱了这首《征兆》。
到最后的副歌，理沙子和鸟一起唱道：
「東の空が白んで、新しい朝が来る」
哭成狗。

31位 向日葵 / 大場、山内、谷

有一段时间向日葵真的很有人气，几场大con都有唱，成员的毕业典礼上也经常出现。

30位 大人の世界 / B・ラヴィエール ☆谷

SF设定的MV倒是挺有趣的。

29位 枯葉のステーション / 菅原

由芽毕业之后这首歌算是彻底没了法定继承人了，原来能唱的或者唱过的S成员也都毕业了个干净，就算是AKB的RH上唱过的乐乐也不在，选择玛雅作为玲奈位置的继承者恐怕也不足为奇，在后来的前のめり里玛雅又继续担任了玲奈位，玛雅回选拔之后还是寄予厚望的。

28位 恋を語る詩人になれなくて / チームS ☆北川綾、アンダー大谷

诗人是稳扎稳打的进了。

27位 MARIA / ☆古畑、内山、日高

MARIA能进也不意外，算是0start里人气比较高的unit了。再加上后面紧接着的MC充分体现了三个人的特色和默契，三个人各有侧重搭配起来又十分和谐，是最高的MARIA了。

27位 無意識の色 / ☆小畑、北川綾、江籠、大場、北野、惣田、高柳、竹内、日高、古畑、鎌田、熊崎、佐藤、末永、菅原、須田

无意识能到这个位置比较意外，大家公认的良曲，但是不知道有这么高的人气。

25位 禁じられた2人 / 矢作、野々垣

我个人还是挺喜欢青G公演的，没想到这次进得这么少（不过估计都用来投梦阶段了）
禁2的版本多种多样，选这两个人出来不无理由，不过我个人来说，更喜欢こはく和天使再这首歌里面的表现。

24位 いい人いい人詐欺 / ☆杉山、☆水野、上村、野島、町、山田、青木、内山、太田、小畑、片岡、白井、松村、相川、浅井、井田、末永、菅原、高畑

好人欺诈也是意外地高！应该是爱理推的作战结果。虽然从头到尾槽点挺多的，但是这首歌还是很可爱的。

23位 ハングリーライオン / 桃：熊崎、緑：鎌田、橙：福士、青：浅井

饿师出来我当时还以为熊推放弃名侦探了呢。
记得剧场实况的时候兰兰解说这首歌歌词背后的内涵，现在想想这首歌真的是有十分深刻的意味。
服装就不吐槽了……
不过偶尔会被这首歌莫名感动一下。

22位 例のポテトチップス / 平田、西、中野、大谷、深井、石黒

研究生们的薯片。
虽然人不多，能走到2日目夜公演已经很不错了。
美滨居然被剪掉了很生气!很想看现场！
其实很好奇这首歌的c是谁。

21位 夕立の前 / 松村、北川綾、北野、竹内、日高、青木、熊崎、山田、鎌田、井田

这次顺位不高，但是6期再zeroposi的live已经达到了一个新高度，猜拳大会加油！朝着zepp努力吧！

20位 誰かの耳 / チームKⅡ ☆小畑

这首歌也不意外。
歌词比较抨击当下的状况，很多人听了应该都有共鸣。
MV和整首歌的风格都比较有特色，也算是K2的集大成作。

19位 前のめり / ☆菅原、北川綾、古畑、井上、岡田、野村、内山、江籠、大場、小畑、惣田、竹内、高柳、古畑、熊崎、斉藤、佐藤、末永、須田、福士

前倾能那么高还是有点始料未及，毕竟玲奈毕业也三年了。
话说回来，前倾的确是一首能让人振奋的好歌。「希望は体力、まだまだ大丈夫」也成为座右铭一样的存在。
玛雅的c算是意料之外情理之中，成员也是准选拔级别的。

18位 Because どっちつかず / コサンガー7 谷、都築、山内、山田、高木、松村

这首歌是我整张22单最喜欢的一首。
如果自己哼歌的话觉得稀疏平常，但是在听的时候就有一种强烈的倾诉的感觉。
行云流水般的歌词也增色不少。
可惜的是没有MV。现场一定很好看吧。

17位 ハートの独占権 / 江籠、高柳

独占权真的很可爱。
这个搭配也真的是绝赞！
给初恋cp点一百个赞！

16位 花の香りのシンフォニー / ☆江籠、北川綾、坂本、野島、荒井、大場、北野、白井、高柳、竹内、鎌田、熊崎、菅原、須田

花香进到这个有点尴尬的位置，我还差点以为天使推放弃奇迹流星群了呢。
话说我到花香的时候才反应过来夜公演有直播……

15位 意外にマンゴー / ☆小畑、北川綾、江籠、大場、北野、惣田、高柳、竹内、古畑、矢作、鎌田、熊崎、佐藤、末永、菅原、須田
